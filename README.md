# DOT 用药时长分析工具

基于网页的销售数据 DOT (Duration of Treatment) 分析工具，用于分析特定时间段内购药患者的用药时长指标。

## 功能特性

- **数据导入**：支持 Excel (.xlsx, .xls) 和 CSV 格式的销售数据底表导入
- **时间筛选**：自定义选择起始月份和结束月份，筛选该时间段内有购药记录的患者
- **DOT 计算**：自动计算去重患者数、累计购药总支数和 DOT 值
- **全量统计**：统计范围覆盖患者从首次购药至最新数据的全部购药记录
- **数据可视化**：
  - 月度购药趋势图（柱状图 + 折线图）
  - 月度 DOT 对比图
  - 患者购药支数分布图
- **明细表格**：支持分页查看和 CSV 导出
- **响应式设计**：兼容桌面端和移动端
- **数据安全**：所有数据在本地浏览器中处理，不上传至服务器

## DOT 计算逻辑

```
DOT = 累计购药总支数 / 去重患者数
```

1. 用户选择时间段（如 2024年1月 ~ 2024年6月）
2. 筛选出该时间段内有购药记录的去重患者
3. 对这些患者，统计其从**首次购药**至**最新数据**的全部购药总支数
4. DOT = 累计购药总支数 ÷ 去重患者数

**示例**：去重患者 10 人，累计购药 100 支，则 DOT = 10

## 使用方法

1. 用浏览器打开 `index.html`
2. 点击「选择文件」上传销售数据底表（Excel/CSV）
3. 确认字段映射（系统自动识别，可手动调整）
4. 选择分析的时间段
5. 点击「开始分析」查看结果

## 字段说明

工具会自动识别以下字段（支持中英文）：

| 字段 | 关键词 |
|------|--------|
| 患者标识 | 会员号、患者ID、患者编号、卡号 |
| 购药时间 | 销售时间、购药时间、购买时间、日期 |
| 购药数量 | 销售数量、购药数量、数量、支数 |
| 患者姓名 | 会员姓名、患者姓名、姓名（可选）|

## 技术栈

- [PapaParse](https://www.papaparse.com/) - CSV 文件解析
- [Chart.js](https://www.chartjs.org/) - 数据可视化图表
